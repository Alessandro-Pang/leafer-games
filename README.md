# Leafer Games

Leafer 小游戏中心，使用 Leafer 开发的小游戏库。

该仓库是为了演示如何使用 Leafer UI 开发小游戏，并提供了一些游戏案例。

游戏案例相对比较简单，可以作为学习参考资料使用。

## 安装

```sh

```

### 开发环境

```sh
npm run dev
```

### 打包

```sh
npm run build
```
